# BLENDENCE - Investor Presentation Website

A modern, professional one-page website for **BUGE GIDA A.Ş.** showcasing the BLENDENCE brand - Turkey's first 100% freeze-dried nutritional mix brand.

## Overview

This website is designed to present the company to potential investors, highlighting:
- Innovative freeze-drying technology
- Comprehensive product portfolio
- Production capabilities and quality certifications
- International expansion plans
- Competitive advantages

## Design Philosophy

**Motto:** "Naturally Powerful, Perfectly Balanced"

The design follows minimalist and modern principles with:
- Generous white space for clarity
- Organic wave elements reflecting the brand
- Clean typography with excellent readability
- Natural color palette inspired by organic nutrition
- Professional layout optimized for investor presentations

## Setup & Usage

### Quick Start

1. **Clone or download** this repository
2. **Open `index.html`** in a modern web browser
3. That's it! No build process required.

### Development

The website uses vanilla HTML, CSS, and JavaScript - no frameworks or dependencies required.

```bash
# Simply open the file in your browser
open index.html

# Or use a local development server (optional)
python -m http.server 8000
# Then visit http://localhost:8000
```

### Customization

1. **Update Content**: Edit `index.html` to modify text and structure
2. **Adjust Colors**: Modify CSS variables in `styles.css` (lines 9-31)
3. **Add Images**: Replace placeholder sections with actual product images
4. **Modify Animations**: Adjust JavaScript in `script.js`

## Product Information

### Adult Products
- **Green Mix**: Natural detox with cucumber, green apple, parsley, spinach, lemon, ginger
- **Red Mix**: Immunity boost with beet, carrot, apple, pomegranate

### Children's Mix Series
- **Ages 4-7**: Growth & immunity focus with sweet fruit profile
- **Ages 8-12**: Mental development & school performance support
- **Ages 13-18**: Energy, concentration & active life support with omega-3

### Technology
- **Process**: Lyophilization (freeze-drying under vacuum)
- **Nutrient Retention**: Less than 5% loss vs. 40-60% in traditional methods
- **Benefits**: No preservatives, extended shelf life, no cold chain required

- Current: ISO 9001, ISO 22000, HACCP

- Future: BRC (UK market entry)
